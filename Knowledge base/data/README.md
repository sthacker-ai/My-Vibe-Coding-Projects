# Generated Data

This folder is for machine-generated indexes that make the Next.js app fast.

Examples:

- `graph.json` for nodes and edges derived from Markdown links, tags, and source
  references.
- `sources.json` for imported source metadata.
- `study-queue.json` for local course progress.

These files can be regenerated from `content/`, so we should treat them as
derived data unless a workflow says otherwise.
