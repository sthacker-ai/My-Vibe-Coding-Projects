# Mastering Claude AI: Breaking Default Mode with Slash Commands

## Overview
This course teaches you how to move beyond Claude's default, passive mode by using specific "slash commands" to force the AI into specialized modes designed for rigorous analysis and strategic output.

## Key Concepts
- Default Mode Problem: The state where Claude agrees with you and produces generic, average responses.
- Slash Commands: Single-word instructions used to break Claude out of default mode and force specific behaviors.
- Steelman: A command that forces Claude to build the strongest, most rigorous defense for a given idea or position.
- Mode Switching: The principle of defining Claude's role (e.g., debate champion) to shift its output quality.

## Chapter 1: The Default Mode Problem
Most users operate Claude in its default mode, which results in output that is merely agreeable and substantively average. This happens because Claude optimizes for feeling helpful rather than being maximally rigorous. To achieve genuinely different, high-quality results, you must instruct Claude on *how* to behave, rather than just *what* to write.

## Chapter 2: The Power of Slash Commands
Slash commands are the solution to the Default Mode problem. By using single-word commands, you install specific behavioral instructions into Claude, directing it into modes optimized for specific tasks, such as debate or rigorous defense. These commands are built once and can be called forever to ensure consistent, high-quality results.

## Key Takeaways
- Recognize that generic output is a setup problem, not a bug, caused by Claude's default mode.
- Use slash commands to permanently change how you use Claude, moving it into specialized modes.
- The `/steelman` command forces Claude to produce the most rigorous, evidence-based defense of your position, moving beyond simple agreement.
- Build your commands once and call them repeatedly to ensure consistent, high-level output.
