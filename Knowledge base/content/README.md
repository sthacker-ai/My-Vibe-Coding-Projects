# KnowledgeBase Content

This folder is intentionally compatible with Obsidian. The Markdown files here
can be opened directly as a vault, or this project can point at an existing
Obsidian vault through environment variables.

## Folders

- `inbox/` holds unprocessed imports and quick captures.
- `sources/` holds one note per original source.
- `wiki/` holds durable concept notes.
- `courses/` holds generated learning paths, modules, quizzes, and reviews.
- `assets/` holds local media we choose to keep.

The source of truth should stay human-readable. The app can build indexes and
graphs from these files instead of locking the knowledge base inside a hosted
database.
